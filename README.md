## 🛠️ Built With

This portfolio is built using modern web development technologies to ensure performance, responsiveness, and a premium user experience:

- **React.js** for building the component-based UI
- **Vite** for fast, high-performance frontend tooling 
- **Tailwind CSS** for utility-first styling and custom dark/light themes
- **Framer Motion** for smooth, hardware-accelerated animations
- **React Router** for seamless client-side navigation

## ✨ Key Features

- **Dark/Light Mode:** Fully customized dual theming.
- **Glassmorphism & Gradients:** Subtle blurs, floating glow elements, and gradient typography for a premium look.
- **Smooth Animations:** Integrated page transitions and element entrance animations.
- **Dynamic Projects Showcase:** A modular data approach (`src/data/projects.js`) makes it easy to add projects without redesigning components.

## ⚙️ Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jeevan-m-21/Portfolio.git
   ```
2. **Navigate & Install dependencies:**
   ```bash
   cd Portfolio
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
